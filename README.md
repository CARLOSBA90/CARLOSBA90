## 📂 Proyectos Destacados

### 🖥️ [Via Terminal](https://github.com/CARLOSBA90)
Utility app de escritorio diseñada para centralizar, gestionar y ejecutar comandos de sistema de forma ágil y nativa.
* **Tech:** Java, CMD/Terminal Integration.
* **Features:** Automatización de flujos por consola, interfaz simplificada y ejecución de comandos básicos.

### 🛒 [Ecommerce API & Front](https://cepr0.com)
Arquitectura de extremo a extremo para la gestión de catálogo, carritos y procesamiento de órdenes de compra.
* **Tech:** Java, Spring Boot, Spring Data JPA, Angular, Docker.
* **Features:** Integración de servicios, persistencia robusta y panel autoadministrable.

### 📊 [Sistema de Punto de Venta (PDV-V2)](https://cepr0.com)
Aplicación full-stack optimizada para la administración de ventas, control de stock y tableros en tiempo real.
* **Tech:** Java, Spring Boot, MySQL, Angular, Docker.
* **Impacto:** Dashboards dinámicos de ventas, emisión de comprobantes y gestión de inventario eficiente.

### 📧 [Mailing Service Microservice](https://github.com/CARLOSBA90/mailing-service-springboot)
Microservicio backend especializado en la orquestación, renderizado y envío automatizado de correos electrónicos.
* **Tech:** Java, Spring Boot, JavaMailSender, REST API.
* **Features:** Soporte para plantillas HTML dinámicas, manejo de adjuntos y alta disponibilidad para correos transaccionales.
